# 🌆 Learning City

**Play-powered mastery.** A 3D open-world educational game where a child's city is built
from their own interests — and every game in it teaches the skills a parent or teacher chose.

▶ **Play it:** https://mordthelord.github.io/learning-city/

- Works in any modern browser (Chrome/Edge/Firefox/Safari), desktop and mobile
- No install, no account — the questionnaire builds your city
- Build r8e — original soundtrack by xNag
